# Learning Evidence Map

This document connects research-supported learning methods to product behavior. It
does not claim that English Coach Local or its placement diagnostic has been
scientifically validated.

## Retrieval Practice

**Evidence:** Roediger, H. L., & Karpicke, J. D. (2006). Test-Enhanced Learning:
Taking Memory Tests Improves Long-Term Retention. *Psychological Science, 17*(3),
249-255. https://doi.org/10.1111/j.1467-9280.2006.01693.x

**Product translation:** Lessons ask learners to retrieve an answer before feedback.
Listening, typing, speaking, and word-building reduce reliance on recognition alone.
Assessment answer positions are balanced and randomized so position patterns cannot
replace retrieval of the language answer.

## Spaced Practice

**Evidence:** Cepeda, N. J., Pashler, H., Vul, E., Wixted, J. T., & Rohrer, D. (2006).
Distributed Practice in Verbal Recall Tasks: A Review and Quantitative Synthesis.
*Psychological Bulletin, 132*(3), 354-380. https://doi.org/10.1037/0033-2909.132.3.354

**Product translation:** Incorrect answers become due for review and correct reviews
move through increasing 1, 3, 7, and 14-day intervals.

## Interleaved Practice

**Evidence:** Rohrer, D., & Taylor, K. (2007). The Shuffling of Mathematics Problems
Improves Learning. *Instructional Science, 35*, 481-498.
https://doi.org/10.1007/s11251-007-9015-8

**Product translation:** Sessions alternate exercise formats and checkpoints sample
multiple categories instead of drilling only one response pattern. Transfer from
mathematics to language learning is a product hypothesis that must be measured.

## Corrective Feedback in Language Learning

**Evidence:** Lyster, R., & Saito, K. (2010). Oral Feedback in Classroom SLA: A
Meta-Analysis. *Studies in Second Language Acquisition, 32*(2), 265-302.
https://doi.org/10.1017/S0272263109990520

**Product translation:** Checked answers receive immediate correctness feedback,
the target form, and an explanation. Speaking practice compares the learner's words
with the target phrase and provides a corrected version.

## CEFR Reference

**Framework:** Council of Europe (2020). *Common European Framework of Reference for
Languages: Learning, Teaching, Assessment - Companion Volume*.

**Product translation:** The pedagogical matrix uses A1-B2 as approximate ranges for
objectives and progressive difficulty. The 12-item placement check is an internal,
criterion-referenced diagnostic. It is not a validated CEFR test or certification.

## Decision Rule

For every major learning mechanic added later, document:

1. The learning problem being addressed.
2. The primary study, systematic review, or established framework supporting it.
3. The exact product behavior derived from that evidence.
4. Any inference made when evidence comes from a different subject or setting.
5. The product metric used to test whether the implementation helps learners.
